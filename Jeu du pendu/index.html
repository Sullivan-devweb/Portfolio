🎮 Jeu du Pendu 🎮


            

            

            
            
            

            

            
        

    


    
    

        

            

                

                    
📜 Règles du jeu

                    
                

                

                    
Comment jouer :

                    
    
                        
  1. Entrez un mot à deviner dans le champ prévu.
    2. 
                        
  2. Cliquez sur "Commencer le jeu" pour démarrer.
    3. 
                        
  3. Devinez les lettres du mot en les saisissant une par une.
    4. 
                        
  4. Vous avez un nombre limité d'essais pour deviner le mot.
    5. 
                        
  5. Si vous devinez toutes les lettres avant d'épuiser vos essais, vous gagnez !
    6. 
                        
  6. Sinon, le pendu est complété et vous perdez.
    7. 
                    

                    
🎉 Bonne chance ! 🎉
